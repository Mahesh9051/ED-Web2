3 ways to use CSS
-----------------
1. inline CSS
>> style="background-color: aqua; color: red "
>> style attribute inside html tags

2. internal CSS
>> <style></style> => tag in <head></head>
 <style>
        h1 {
            background-color: aqua;
            color: red
        }
    </style>

 3. external CSS
>> <link> tag calling inside head tag
>> <link rel="stylesheet" href="styles.css" />

Block element
----

>> block level element
>> start on the new line
>> occupy whole width
>> eg: div, p , heading tags

Inline element
---------------

>> Start in the same line
>> only take space which is required
>>eg: span, img, a, audio, formatting tags



display: none and visibility: hidden difference?
--------------------------------------------------

display: none  => will not occupy space
visibility: hidden => will occupy space

display: block & visibility: visible => text will be visible
